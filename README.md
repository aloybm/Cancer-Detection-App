# Skin Cancer Detection App
App to share and discover stories from every users.

## Features 
- Register and Login
- Add and Share Story
- Story Location

## Tools
- TFLite Model
- Save history in Room
- uCrop for Image Cropping
- ViewPager2 for Cancer News View
